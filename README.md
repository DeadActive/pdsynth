# PD Synth — Синтезатор с фазовой дистортией

Браузерный синтезатор, реализующий метод синтеза **Phase Distortion (PD)** — технику, разработанную Casio в 1984 году для серии синтезаторов CZ. Суть метода: нелинейная функция-«искажатель» читает форму волны с переменной скоростью внутри одного периода, изменяя тембр без изменения высоты тона.

## Как пользоваться

1. **Нажмите «Click to Start»** — это инициализирует Web Audio API (браузер требует жест пользователя перед воспроизведением звука).

2. **Играйте ноты** клавишами мыши на виртуальной клавиатуре в нижней части экрана, либо с компьютерной клавиатуры:

   | Клавиши (белые) | A S D F G H J K |
   |-----------------|-----------------|
   | Клавиши (чёрные) | W E — T Y U |

3. **VCO 1 и VCO 2** — два независимых осциллятора. Для каждого можно настроить:
   - **Форму волны** — Sine / Saw / Square / Triangle
   - **Level** — громкость осциллятора
   - **Mod** — глубина фазовой дистортии (от 0 = чистая форма волны до 0.99 = максимальное искажение)

4. **ENVELOPE** — огибающая ADSR управляет атакой, спадом, уровнем сустейна и временем затухания.

5. **SCOPE** — осциллоскоп с синхронизацией по частоте ноты:
   - **OCT +/−** — зум: смещение на октаву изменяет количество отображаемых периодов
   - **Color** — цвет луча (вращение по цветовому кругу)
   - **Blur** — размытие / свечение в стиле фосфорного экрана
   - **Smooth** — сглаживание кривой (усреднение семплов)

## Принцип работы Phase Distortion

Линейная фаза φ ∈ [0, 1] преобразуется кусочно-линейной функцией с точкой излома в (x_j, 0.5):

```
φ < x_j  →  φ' = φ · 0.5 / x_j
φ ≥ x_j  →  φ' = 0.5 + (φ − x_j) · 0.5 / (1 − x_j)
```

При **Mod = 0** → x_j = 0.5 → прямая линия, искажения нет.  
При **Mod → 1** → x_j → 0 → первая половина периода читается мгновенно, создаётся богатый обертонами звук.

Искажённая фаза φ' затем подаётся на генератор выбранной формы волны (синус, пила, квадрат, треугольник).
